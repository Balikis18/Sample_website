Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous In the process of making it, we will be learning how to make HTML tables via table and related tags

The table below dont include all HTML tags but the do list many of the most important ones

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (level1–6)
P paragraph include a paragragh of text
table table include a table
tr table row include a row of data
th Table header Make a table header
td Table data Include a table data cell
header header wrap content for styling
div div wrap content into section

Document tags

Tag Name Purpose
html to wrap content for html markup
head head section for pages, titles, scripts and other meta information
body body contains all visible contents of website
title title to specify the title of a webpage
meta meta for specifiying info that helps with better rendering or treatment of a web content
div div to divide content
span span make content inline with surrounding content
blockquote blockquote include quotation
header header isolate the head text
br break break the line

Block element

Inline Element

Tag Name Purpose Example Result
em emphasized to emphasize or italize text <em > technical sophistication </em > technical sophistication
Strong bold to bold text <strong >at least a billion people </strong > at least one a billion people
a anchor hyperlink <a href="https://www.learnenough.com" >Learn enough </a > learn enough
IMG image add image <img src="images/small_twitter_logo.png" alt="Learn Enough Twitter Logo">
code code to display piece of markup <code >the table below </code > the table below
span span wrap content for styling <span >call me Ishmeal </span > call me Ishmeal
